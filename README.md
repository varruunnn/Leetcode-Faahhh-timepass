## 🚀 How to Install (Free / Developer Mode)

Since this is an open-source project and not hosted on the Chrome Web Store, you can install it for free in about 30 seconds using Chrome's "Developer Mode."

### Step 1: Get the Code
1. Click the green **`<> Code`** button at the top of this repository and select **Download ZIP**.
2. Extract the downloaded `.zip` file into a folder on your computer (e.g., your Desktop or Documents).

### Step 2: Load it into your Browser
This works on Google Chrome, Microsoft Edge, Brave, and most Chromium-based browsers.

1. Open your browser and type the following into your URL bar:
   * **Chrome / Brave:** `chrome://extensions/`
   * **Edge:** `edge://extensions/`
2. In the top right corner, turn on the toggle switch for **Developer mode**. 

3. In the top left corner, click the **Load unpacked** button.

4. Select the folder where you extracted the code in Step 1. 

**Boom. You're done.** The extension should now appear in your list of active extensions.
